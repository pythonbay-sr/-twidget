1. Open OBS and create a new browser source
  2. Set the browser source URL to localhost:3000/source
  3. Set the width to 450 and the height to 500
  4. Insert the custom css: body { background-color: rgba(0, 0, 0, 0); margin: 0px auto; overflow: hidden; }
  5. Check the "Refresh browser when scene becomes active" option
  6. Click OK

  1. Open Twidget and sign in if you have not
  2. Open OBS and open the scene where you have set up the Twidget browser source
  3. Click the Start button and enter the livestream URL found in the address bar. (make sure to include the HTTPS prefix)
  4. You can also connect to a Facebook livestream by pasting the Facebook livestream URL to Twidget once you have logged in with Facebook.
  5. Wait until the stream chat comments get loaded in the chatbox
  6. Select the specific comments you would like to display in OBS
  7. The selected comments will appear in the OBS modal, this may take a while at the beggining. If it still does not appear after one minute, refresh the scene
  8. You can also unselect comments you have already selected
  9. Scroll through the chatbox to see all the comments, Twidget updates the chatbox every 5 seconds
  10. You can change the theme of the chat source that appears in OBS by clicking the settings icon in the Chatbox Menubar and choose the templates
  11. To stop the stream, press the stop button
